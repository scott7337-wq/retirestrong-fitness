# RetireStrong Fitness

Professional fitness tracking app for adults 60+

## Quick Start

npm install
npm start

## Features

- AI-powered coaching
- Progress tracking with charts
- Body measurements & photos
- Pain tracking
- Nutrition guidelines
- Achievement system
- Exercise library
- Export to CSV

## Tech Stack

- React 18
- Recharts
- Lucide React
- localStorage

## Deploy

Push to GitHub and deploy via Vercel
